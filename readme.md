# 🌐 Mi Portafolio Personal Web

¡Bienvenido/a a mi repositorio del portafolio! Este es un espacio interactivo diseñado y desarrollado desde cero para dar a conocer mi trayectoria, mostrar mis habilidades técnicas como desarrollador y centralizar los proyectos de software en los que he trabajado.

La interfaz cuenta con un diseño oscuro moderno (*dark mode*), efectos translúcidos (Glassmorphism), animaciones fluidas y es completamente responsiva para cualquier dispositivo móvil.

## 🛠️ Tecnologías y Herramientas Utilizadas

Para el desarrollo de este proyecto se implementó un flujo de trabajo moderno en el frontend utilizando:

* **HTML5** - Estructuración semántica y accesible del contenido.
* **CSS3 Avanzado** - Diseño visual mediante CSS Grid y Flexbox, variables globales, efectos de resplandor (*glow effects*) y adaptabilidad móvil (*Media Queries*).
* **JavaScript (ES6+)** - Manipulación del DOM, efectos dinámicos en la barra de navegación sincronizada con el scroll mediante `IntersectionObserver`.
* **React** - Integración de micro-componentes renderizados en el cliente (como la animación flotante del Hero).
* **Remix Icons** - Biblioteca de iconos vectoriales utilizada para el diseño de tarjetas y redes sociales.
